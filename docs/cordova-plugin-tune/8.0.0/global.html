isTuneLink(appLinkUrl, success, failure)

Test if your custom Tune Link domain is registered with Tune.

Tune Links are Tune-hosted App Links. Tune Links are often shared as short-urls, and the Tune SDK will handle expanding the url and returning the in-app destination url to didReceiveDeeplink registered via registerDeeplinkListener.
Parameters:
Name Type Description
appLinkUrl string url to test if it is a Tune Link. Must not be null.
success callback Callback returns true if this link is a Tune Link that will be measured by Tune and routed into the TuneDeeplinkListener. false if not.
failure callback Callback Message if there was an error.
